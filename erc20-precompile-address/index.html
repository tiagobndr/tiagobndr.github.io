How it works

Correct Pattern for Paseo Asset Hub:

Format: 0x + assetId (8 chars) + zeros (24 chars) + 01200000

The 0x120 suffix indicates a Trust Backed Asset prefix on Polkadot Hub Testnet.

Example: Asset ID 50000281 → 0x02Faf19900000000000000000000000001200000
